01Executive Summary
This repo is an active typing-performance trainer plus a separate STRATA study-tracker design-system prototype.

The main app is a dark HUD-style typing trainer with modes, pause behavior, analytics, history, and keyboard diagnostics. The STRATA files are not wired into the app, but they provide a stronger token architecture and a possible future product direction around missions, ranks, XP, and learning progression.

08Recommended Token Relayout

Use STRATA's two-tier structure, adapted to the typing app: primitives for raw values, semantic aliases for UI intent, CSS variables for Tailwind and shadcn.

primitives.color.void950      = #09090B
primitives.color.signal500    = #F97316
tokens.color.surface.app      = primitives.color.void950
tokens.color.signal.active    = primitives.color.signal500
tokens.color.signal.chart     = primitives.color.dataBlue400
tokens.color.signal.error     = primitives.color.danger500
StepChange
1Add src/design/tokens.ts.
2Add src/design/theme.css for CSS variables.
3Map src/index.css theme values to semantic variables.
4Replace repeated orange/zinc utility classes in core components.
5Keep or migrate STRATA as a reference system.

10Naming Suggestions
Recommended name: Keystation. It keeps the console/workstation feel, references keyboard practice, and can grow beyond typing speed.
NameFit
KeystationBest all-around name for a typing and training console.
TypefieldGood if STRATA's field-research metaphor becomes central.
GlyphrunMore game-like; good for streaks, ranks, and challenges.
SignaltypeStrong fit for HUD/telemetry visual language.
StrataTypeBest if STRATA learning progression merges with typing practice.
CYBERTYPEKeep only if the app stays intentionally cyberpunk and typing-only.
